# FastAPI UI Specs

This directory is the home for UI-related specifications used by the FastAPI PWS Admin Console.

## Specs

- [PWS Admin Tab/Subtab Specification](pwsadmin-tab-subtab-specification.md)
  - Registry-driven dashboard tab and subtab contract.
  - Template, CSS, JavaScript module, URL activation, and test requirements.

- [Platform Onboarding Specification](platform-onboarding-spec.md)
  - Platform metadata contract for API tokens.
  - Platforms tab token-slot behavior, validation, and runtime connector requirements.

- [Platform Listing List UI Specification](platform-listing-list-ui-spec.md)
  - Listing-list behavior for platform fetch/import UI.
  - PriceLabs/Wheelhouse shared-coordinate parity, duplicate/highlight rules, link annotations, and import behavior.
