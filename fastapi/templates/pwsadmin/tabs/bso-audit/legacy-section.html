Help: how to use BSO Audit

booking_entry_id: booking ID from booking_registers.id.

status: if empty, the list shows only applied and removed.

updated_from / updated_to: date range filter on updated_at (inclusive by day).

Instruction: shows a short excerpt; click it to open full instruction in YML format.

Updated / Applied / Removed: shown in your browser's local timezone.

Next: loads older rows using cursor pagination.
