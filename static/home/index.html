OAuth 2 Authorization for Automation

This landing page powers secure OAuth 2 authorization on auto.palmwavestays.com/hom for Palm Wave Stays integrations. Use it to connect your automation platform while keeping credentials protected.

Connection Details

Use the values below when configuring n8n or another OAuth client.

Redirect URL https://auto.palmwavestays.com/rest/oauth2-credential/callback
Webhook URL https://auto.palmwavestays.com/webhook/
From n8n env N8N_EDITOR_BASE_URL=https://auto.palmwavestays.com
From n8n env WEBHOOK_URL=https://auto.palmwavestays.com/webhook/

Why this page exists

Palm Wave Stays automations use OAuth 2 so third-party tools can access only what you approve. This page confirms the official URLs, branding, and purpose for the auto.palmwavestays.com OAuth flow.

No bookings or payments are created from this page. It only supports secure authorization.

Authorization steps

  1. Select "Connect Palm Wave Stays" inside your automation tool.
  2. Sign in and approve access on the Palm Wave Stays consent screen.
  3. Return to your tool to complete the connection and save tokens.

Security notes

  • Tokens are stored by your automation tool, not on this page.
  • Access can be revoked at any time from the tool settings.
  • We request only the permissions needed for the workflow.
